# PDFs Oficiales – Normativa AGF Chile

Esta carpeta está destinada a contener los **PDFs oficiales** publicados por la Comisión para el Mercado Financiero (CMF) y la Biblioteca del Congreso Nacional (BCN) de Chile para cada norma del marco regulatorio de las AGF.

## Cómo descargar los PDFs

Ejecuta el script desde la raíz del repositorio:

```bash
bash descargar_normativa.sh
```

El script descargará automáticamente todos los archivos en esta carpeta.

## Por qué los PDFs no están en el repositorio

Los PDFs son documentos oficiales publicados por la CMF. Para garantizar que siempre se utilice la **versión vigente y actualizada**, se recomienda descargarlos directamente desde la fuente oficial en lugar de almacenar copias estáticas.

Si necesitas una copia local para uso offline o para contexto de IA, ejecuta el script `descargar_normativa.sh`.
